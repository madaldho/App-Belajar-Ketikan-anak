# Product Overview

**YukMengetik** is a typing speed test application designed for Indonesian school competitions with kiosk mode support.

## Core Purpose
Professional typing test application for educational environments, featuring real-time per-character feedback similar to MonkeyType.

## Key Features
- **Per-character typing mode** with real-time green/red feedback
- **Student mode** with class selection and leaderboard integration
- **Admin panel** for managing students, classes, and test sentences
- **Kiosk mode** with auto-return to home and touch-friendly interface
- **Leaderboard system** with rankings and statistics
- **Offline-ready** using SQLite database

## Target Users
- Students (primary users taking typing tests)
- Teachers/Admins (managing students, classes, and viewing analytics)

## Language
All UI, content, and documentation is in **Bahasa Indonesia**.
